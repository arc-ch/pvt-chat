# >private-chat

This is a realtime chat application built with Next.js, React, and TypeScript.

## Features

- **Private chat rooms**: Users can create and join private chat rooms to have real-time conversations with their friends.
- **Real-time messaging**: Messages are sent and received instantly, providing a seamless chat experience.
- **Room creation**: Users can create new chat rooms by providing a unique room ID.
- **Room destruction**: Chat rooms can be destroyed by the creator, deleting all messages and disconnecting participants.
- **User authentication**: Users must authenticate with a unique username to join a chat room.


## Tech Stack

- **Next.js**: A React framework for building server-rendered and static web applications.
- **React**: A JavaScript library for building user interfaces.
- **TypeScript**: A statically typed superset of JavaScript that adds types to the language.
- **@elysiajs/eden**: A realtime database for building real-time applications.
- **@tanstack/react-query**: A library for managing and caching API data in React applications.
- **@upstash/realtime**: A realtime messaging library for building real-time applications.
- **@upstash/redis**: A Redis client library for interacting with Redis databases.
- **date-fns**: A library for handling and manipulating dates and times.
- **nanoid**: A library for generating unique identifiers.
